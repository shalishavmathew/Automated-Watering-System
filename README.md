# Automated-Watering-System
This was a project that was completed in January of 2026. The goal of this project was to design a system that detects moisture inside of plant soil and waters it if it doesn't meet an intended minimum, a water pump activates to maintain it at it's minimum value of moisture. This involved various testing of different moistures in soil to find the intended value necessary, testing individual components to ensure that it could be incorporated into different circuits, testing circuits with multimeters to ensure voltages and currents were at intended values.
